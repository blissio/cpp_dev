## What's in here

This repo is my active C++ workspace — coursework, experiments, and implementations I built to actually understand how things work under the hood.

### Notable implementations

| File | Description |
|---|---|
| `BST.cpp` | Binary search tree — insert, search, traversal |
| `stacks.cpp` | Stack implementation from scratch |
| `list.cpp` / `pointer_list.cpp` | Linked list variants using raw pointers |
| `vectors.cpp` | Dynamic array / vector internals |
| `Fraction.cpp` + `Fraction.h` | Fraction class with operator overloading |
| `opOverloading.cpp` | Custom operator overloading patterns |
| `LargestFirstColoring.cpp` | Graph coloring algorithm |
| `C-String.cpp` | Manual C-string manipulation without STL |

## Concepts covered

- Pointers, references, and manual memory management
- OOP — classes, constructors, operator overloading
- Data structures — BST, linked lists, stacks, dynamic arrays
- Algorithm implementation from scratch
- Header/implementation separation (`.h` / `.cpp`)

---

## Why C++

C++ sits at the intersection of performance and abstraction. Understanding it at this level — raw pointers, manual memory, no garbage collector — is foundational for systems programming, game development, and security research (reading disassembly is a lot easier when you wrote similar code yourself).
